D-Bird

Citizen science

Client: New York City Audubon Society

Active demo

Every year, millions of birds are killed by colliding with buildings - this is a major conservation issue that NYC Audubon is working to address. D-Bird is a fully-responsive mobile and desktop web app created for NYC Audubon to complement their research and advocacy efforts by collecting crowdsourced georeferenced bird mortality data from citizen scientists.

D-Bird has recently launched in Texas, Minnesota, and Atlanta.

D-Bird is built in a LAMP stack and includes an adminisatrator portal with data visualization, summary, and export functions.

App design by Patrick Harry - patrickharry.com

Stripe/AgileCRM Handler

Customer management

Client: Circles

Video demo

Circles manages their customer base using AgileCRM, but payments are handled by Stripe.

This app serves as an interface between the two services, working with their REST APIs to first create a customer object in Stripe and then add that customer info to the user's AgileCRM profile.

The app was built on an Express server in the NodeJS framework.

Project Safe Flight Survey Tool (PSFST)

Dynamic field data collection form

Client: New York City Audubon Society

Active demo

Created for NYC Audubon's scientific bird mortality study, Project Safe Flight, PSFST is an efficiency solution that allows project volunteers to record up to 360 entries per form submission - it was designed to replace a Google Form, which only allowed one entry per submission.

PSFST is built in a LAMP stack.

Birding by Subway

Custom Google Map

Client: New York City Audubon Society

Active demo

New York City has some of the best birding locations in the United States, and also one of the best public transit systems! This map provides an overview of some of the City's top birding spots and allows users to quickly get public transit (or other) directions.

Birding by Subway is built in a LAMP stack with the Google Map API.

GeoJSON Doodler

Map geometry drawing and export tool

Click here for demo

GeoJSON Doodler allows users to draw and add attributes to their own custom map geometries. The geometries can then be exported as individual GeoJSONs or bundled into a .csv file, ready for use in other mapping platforms.

Demo not available here, please visit http://field-set.com/geojson_doodler/ to try it out.
