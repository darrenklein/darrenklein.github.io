D-Bird

Crowdsourced data collection

Client: New York City Audubon Society

Active demo

Every year, millions of birds are killed by colliding with buildings - this is a major conservation issue that NYC Audubon is working to address. D-Bird is a fully-responsive mobile and desktop web app created for NYC Audubon to complement their research and advocacy efforts by collecting crowdsourced georeferenced bird mortality data.

In addition to the data collection tool, each client has access to an administrator portal that includes data visualization, summary, and export functions.

D-Bird has recently launched in Texas, Minnesota, and Atlanta.

Project Safe Flight Survey Tool (PSFST)

Dynamic field data form

Client: New York City Audubon Society

Active demo

Created for NYC Audubon's scientific bird mortality study, Project Safe Flight, PSFST allows project volunteers to record up to 360 entries per form submission.

PSFST is built on a PHP/MySQL back-end.

GeoStudy

Field data collection suite

GeoStudy is a prototype field research data collection suite, modelled on a fictional food distribution and purchasing study.

Built in Ruby on Rails, GeoStudy allows researchers to create georeferenced study sites and collect various types of related field data.

A dedicated administrator portal allows project leaders to dynamically define study parameters, review survey progress, and export summary field data reports.

Long Island Volunteer Alewife Survey (LIVAS)

Field data collection

Client: Seatuck Environmental Association

Active demo

Created for Seatuck, LIVAS is a fully-responsive mobile and desktop web-app designed to collect volunteer-sourced georeferenced river herring population survey data.

In addition to the data collection tool, the client has access to an administrator portal that includes data summary and export functions.

LIVAS is built on a PHP/MySQL back-end with a Zurb Foundation responsive front-end.

Carto Doodle

Map geometry input and editor

Active demo

Carto Doodle allows users to draw, edit, and delete their own custom map geometries - try it out and add a few! After submitting, click the link to go to the admin tool to see your work.
